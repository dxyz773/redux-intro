## Instructions To Start App:

1. Fork Redux Intro repo
2. Git clone forked copy to local environment and open in IDE/code editor
3. Open a new terminal in IDE/code editor
4. Install dependencies:

```
  npm i
```

5. Start App

```
  npm start
```
